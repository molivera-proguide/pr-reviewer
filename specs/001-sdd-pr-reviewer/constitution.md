# Constitution

1. Read-only is a product invariant, not a configuration preference.
2. Deterministic checks establish identity, paths, revisions, limits, evidence, status, and verdict thresholds.
3. Models analyze bounded untrusted data and cannot enlarge their capabilities.
4. Every external boundary is validated and every long operation is cancellable.
5. Absence of analysis is reported as incomplete, never as compliance.
6. Credentials and repository content are absent from logs; reports are local and temporary.
7. GitHub and GitLab implement one provider-neutral contract.
8. The compiled binary is the acceptance artifact.
