# Feature plan

The domain contracts and security kernel are implemented first. MCP and provider adapters depend on that kernel. SDD context feeds isolated structured model roles, followed by deterministic evidence verification, conservative verdict calculation, and a private report. Compiled artifact and stdio conformance tests close acceptance.

See `PLAN_IMPLEMENTACION.md` for the full architecture, limits, risk register, and phased gates. ADR 001 records the toolchain decision; `docs/security-model.md` records the threat model.
